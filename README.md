# 数字人生（Digital Life）

> 一个用于 **记录、可视化与分析个人数字人生数据** 的 Obsidian 插件。

**数字人生** 插件旨在将你在 Obsidian 中的结构化记录，转化为 **可视化图表、时间线与关系视图**，帮助你从“零散记录”走向“可分析的人生数据”。

---

## ✨ 核心理念

- 📌 **记录优先**：不改变你的记录习惯，基于 Markdown 数据工作  
- 📊 **可视化呈现**：将任务、事件、进度转化为直观图表  
- 🧠 **结构化分析**：关注“事实与进展”，而非主观反思  
- 🧩 **模块化设计**：插件内部高度解耦，便于维护与扩展  

---

## 🚀 已实现功能

### 1. 数据解析与结构化
- 解析 Obsidian 中的 Markdown 文件
- 支持从文件、内容、表格中提取结构化数据
- 将原始文本转化为统一的数据模型供渲染层使用

### 2. 多种图表渲染组件
插件内置多种可视化渲染器，包括：

- 📊 **柱状图（Bar Chart）**
- 📈 **折线图（Line Chart）**
- 🔵 **散点图（Scatter Chart）**
- 🕸 **雷达图（Radar Chart）**
- 🔥 **热力图（Heatmap）**
- 🥧 **饼图（Pie Chart）**
- 🫧 **气泡图（Bubble Chart）**
- 🌳 **Git Line / 关系演进图**
  - 类似 Git 提交历史的分支/收束可视化
  - 支持任务树、阶段演进、完成状态展示

### 3. 图表交互能力
- Hover 高亮节点 / 父链 / 子树
- 支持复用轨道（lane reuse），避免视觉冗余
- 动态尺寸计算，适配不同视图宽高

### 4. Widget 化 UI 架构
- 所有图表以 Widget 形式存在
- 支持在插件主界面中自由组合
- 为后续 **布局编辑 / 拖拽 / 个性化配置** 打下基础
